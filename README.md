# 🛒 Flask Products API

A simple **Flask-based REST API** for managing products.  
Supports `GET`, `POST`, and filtering products by price.

---

## 🚀 Features
- Get all products
- Get single product by ID
- Filter products by minimum price
- Add new product (with authentication)

---

## 📦 Tech Stack
- Python 3.x
- Flask
- Postman (API testing)
